# LinkedIn Poster with Qwen Content Generation

## ✅ What Changed

The LinkedIn Poster workflow now uses **Qwen Code** to generate unique, engaging content instead of using basic templates.

---

## How It Works

### Before (Template-Based):
```
🚀 Exciting Update!

About Agentic AI

Key highlights:
✅ New feature available now
✅ Improved user experience
✅ Better performance and reliability

#ProductUpdate #Innovation #AI
```

### After (Qwen-Generated):
```
🤖 Agentic AI is revolutionizing how we automate complex workflows. 

From autonomous decision-making to adaptive learning, these intelligent agents are reshaping industries worldwide. 

The future of work is here, and it's powered by AI that takes initiative. 

#AgenticAI #Automation #FutureOfWork #AI #Innovation
```

## New Features

### ✅ Unique Content Every Time
- Qwen generates different content for each post
- No repetitive template language
- Fresh perspectives on your topics

### ✅ Exactly 40-60 Words
- Concise and engaging
- Perfect for LinkedIn's algorithm
- Easy to read on mobile

### ✅ Relevant Emojis & Hashtags
- 2-3 professional emojis
- 3-5 relevant hashtags
- Auto-extracted from generated content

### ✅ Fallback to Template
- If Qwen unavailable, uses templates
- Always generates content
- No failures

## Qwen Prompt Used

```
Create a professional LinkedIn post about: {topic}

Requirements:
- Exactly 40-60 words (short and engaging)
- Professional tone
- Include 2-3 relevant emojis
- Include 3-5 relevant hashtags at the end
- No placeholder text like "[Your link here]"
- Make it sound authentic and engaging

Format the post as plain text (no markdown).
```

## Fallback Behavior

If Qwen Code is unavailable (timeout, not installed, etc.):

```
⚠️  Qwen timed out, using template
📝 Post generated and saved to: ...
```

The workflow **always completes** - it just uses templates as a backup.

## Troubleshooting

### "Qwen not found"
**Solution:** Make sure Qwen Code is installed and in PATH

### "Qwen timed out"
**Solution:** Increases timeout or use simpler topics

### "Using template"
**Normal fallback** - workflow still completes successfully
